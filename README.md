# Calculadoras de Modelado y Simulación

Proyecto Streamlit para resolver y visualizar temas de la materia.

## Ejecutar

```powershell
pip install -r requirements.txt
streamlit run app.py
```

## Deploy en Streamlit Community Cloud

1. Subir este proyecto a GitHub.
2. Entrar a `https://share.streamlit.io`.
3. Crear una app nueva desde el repositorio.
4. Configurar:
   - Branch: `main`
   - Main file path: `app.py`
5. Deploy.

## Tema disponible

- Sistemas lineales no homogéneos: preservación o ruptura de comportamientos.

La estructura está preparada para agregar más temas dentro de `src/topics`.
